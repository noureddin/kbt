(English is below.)

يبدأ معك هذا البرنامج من الدرس الأول بتدريبك على طباعة كلمات كاملة، وليس سلاسل لا معنى لها من الحروف مثل أكثر البرامج الأخرى. فهذا يجعل التدرب أمتع وأفيد، لأنك لا تتعلم فقط أماكن الحروف ولكنك تتعلم أيضا تركيبات الحروف، أي تتدرب على كتابة حرفين أو ثلاثة معا. وهذه مهمة جدا لسرعة الطباعة وصحتها.

لا تحتاج إلى تثبيت لوحة المفاتيح التي تريد تعلمها على نظامك؛ بل تستخدم أي تخطيط لوحة مفاتيح (حتى لو لم يكن عربيا) وسيتصرف البرنامج كأنك تستخدم اللوحة المطلوبة.

هذا البرنامج مستوحى من Learn.Dvorak.NL. بل إنه صُمم في الأصل ليكون نسخة أراك منه.

تحذير: استخدام هذا التطبيق سيجعلك لا تستطيع تقبل أي برنامج يمرنك على حروف متكررة وليس كلمات، أي جميع برامج تمارين الطباعة الأخرى. ؛)

This starts from the very first lesson by training you to type full words, instead of nonsensical strings of letters like most other software. This makes it more fun and rewarding, because you not only learn the positions of the letters, but you also learn letter combinations, which is important for the typing speed and accuracy.

You don't need to install any keyboard layouts; use your normal keyboard (even if the language is different) and this app will behave as if you're using the keyboard you're learning.

This web app is primarily inspired by Learn.Dvorak.NL. In fact, it's made at first to be an Arak clone of it.

Warning: Using this app will spoil you and you will never be able to tolerate any keyboard training software that trains you on typing repetitive letters but not full words, which is to say, all of them. ;)
